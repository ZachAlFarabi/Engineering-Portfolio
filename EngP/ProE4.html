Project E.04

WARMAN 2023 Competition



● Objectives

The WARMAN Competition is a comptetion to create a robotic solution to a given physical and mechanical problem scenario. This required the collaboration of a team, in which integration of software, electrical, and mechanical components was important to conduct solution.

Go back!

● Technical Details

  • Designated programmer for the team, creating closed-loop motor solutions based on time-of-flight sensor readings. This was programmed using Arduino.
  • Led the integration of electrical components, in which knowledge in electronics was imperative to lead the team. This included connections of the motors, motor controllers, sensors into the main Arduino PCB.
  • Aided the design and technical drawings for mechanical components for manufacturing using a water-jet cutter. This was done using 3DExperience CATIA.
