Project P.03

Aircraft wing structures data modelling





● Objectives

As part of one of my required laboratories for Aerospace Engineering, an aircraft wing structures analysis laboratory was required. Though this laboratory was largely an Engineering and mathematical analysis, this laboratory allowed me to further enhance my abilities to process and create databases in MATLAB, using manipulation of arrays, tables, external CSVs, and functions. This allowed me to create required advanced visualisations on MATLAB.

Go back!

● Technical Details

  • Using principles of Velocity-load-factor (Vn) theory, compute a set of parameters and assumptions, of whihc then using advanced plotting techniques, create the Vn-diagram of a Mooney M20 aircraft. This also allowed me to further learn the large theories that is involved in aircraft structures in a velocity's perspective.
  • Using principles of Anderson's line-lifting theory, use discretisation methodologies and create datastructures of 2D-approximation of lift of a Cessna-152 aircraft wing.
  • Using principles of stress analysis and design and complex functions, create piecewise processes to be applied to a dataset of loading on an experimental beam structure.
  • Manipulate datastructures as a whole, which includes arrays, tables, CSV, and with the use of functions and computation methods. This also is also involved with advanced plotting techniques.
