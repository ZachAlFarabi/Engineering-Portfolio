Project P.05

PDF Grid-space Note-taking Compiler





● Objectives

The purpose of this project was to develop my technical skills in PDF processing and automation by creating a Python script that merges multiple pages of a PDF into a single vertically stacked page while maintaining a consistent A4 aspect ratio. This was achieved by extracting page images, combining them dynamically, and adding a calculated blank grid space on the right side for annotations. The project also incorporated optional functionality to append extra blank grid pages at the end of the output document. This was done using libraries such as PyMuPDF, ReportLab, and PIL to handle page rendering, layout, and image processing.

Go back!

● Technical Details

  • Merge multiple pages vertically into one combined PDF page by iterating through the source PDF and stacking page pixmaps.
  • Dynamically calculate and append blank grid space to match a target aspect ratio (A4) and draw a light grey grid using ReportLab.
  • Export and re-embed remasterised page images with correct offsets using PIL and temporary PNG files for each page section.
  • Optionally add extra blank grid pages at the end of the output PDF, maintaining consistent page size and grid alignment.
