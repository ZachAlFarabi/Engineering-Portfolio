Option Derivative Function Model





● Objectives

The purpose of this project was to expand my financial modelling skills and knowledge for evaluating a binomial model for European and American Potions, and comparing the differences between the two options. This works for the two kinds of options; calls and puts. This was done using MATLAB as a means of evaluating the options as matrices. This was attempted to be expanded to include barrier options, however is not finished.
